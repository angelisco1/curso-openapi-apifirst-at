Peticiones relacionadas con usuarios

Obtiene los usuarios

Obtiene los usuarios, por defecto una lista de 10 usuarios, y puedes utilizar los parámetros page y limit para traer más usuarios y otras páginas.

query Parameters
page
integer
Default: 1
limit
integer
Default: 10

Responses

Servidor local

http://localhost:3000/usuarios

Servidor de desarrollo

http://development.miapi.com/{version}/usuarios
